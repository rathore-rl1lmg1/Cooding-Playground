# Cooding-Playground

## collection of my learning-based implementations across robotics, reinforcement learning, control systems, and general programming.
This repository serves as a hands-on experimentation space where I test ideas, build intuition, and implement concepts from scratch.

## Deep Learning
This section of Coding Playground focuses on building a strong foundation in Deep Learning (DL) by implementing models from scratch and using modern frameworks.
The primary goal is to understand how neural networks work internally before leveraging tools like PyTorch.

## Reinforcement Learning
This repository contains my implementations of Reinforcement Learning (RL) algorithms.
The focus is on building algorithms from scratch to deeply understand their working principles and behavior.

## Deep Reinforcement Learning
This section of Coding Playground focuses on implementing Deep Reinforcement Learning (DRL) algorithms using neural networks.
The objective is to understand how reinforcement learning scales with deep learning by combining value-based and policy-based methods using PyTorch.
